# Social Media App

This is a general purpose social media app

## Key features of the App

- User Profile
  - Profile Page
  - Login mechanism
- User generate posts
  - Feed
  - Post creation page
- Followers
  - Visit profile pages using readonly mode
- Like & comment system

## Tech Stack

- Next.js with App Dir
- Next Auth
- Prisma
- Postgres
- Shadcn UI
